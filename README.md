# Epul Portfolio Website

Modern portfolio website for **Saepul (Epul)** — Photographer, Videographer, and Content Creator.

## Overview

This project is a personal branding portfolio website built to showcase:

* Professional profile
* Experience
* Skills
* Contact information

The design uses a modern dark theme with futuristic accents and responsive layout.

## Tech Stack

* HTML5
* CSS3
* Google Drive Thumbnail Image
* Nginx
* Cloudflare Tunnel

## Features

* Responsive design
* Animated skill power bars
* Modern glassmorphism UI
* Social media integration
* Fast lightweight static deployment
